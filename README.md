Добро пожаловать в захватывающий мир приключений и тайн!

Цель игры проста, но требует смекалки и удачи: найдите спрятанные сокровища, преодолевая препятствия и ловушки, расставленные на пути. Игрок проходит уровни, решает загадки и собирает артефакты, чтобы приблизиться к победе.

Игра построена на модульной структуре, обеспечивающей удобство разработки и расширения функционала:

- Модуль констант: содержит базовые настройки и важные игровые переменные.
- Модуль действий игрока: реализует ключевые функции взаимодействия с уровнями и предметами, позволяя игроку собирать бонусы и решать головоломки.
- Модуль утилит (utils): включает вспомогательные функции, необходимые для общей функциональности игры.
- Модуль main: основной файл, управляющий ходом игры и координацией остальных компонентов.

Каждый сеанс заканчивается победой (успешное получение сокровищ) или поражением (ловушка или ошибка). По завершении игрок возвращается к началу приключения, открывая новые возможности и стратегии прохождения.

Инструкция по игре

1. Скачать репозиторий.
2. Установить зависимости (через - poetry install).
3. Запустить игру командой poetry run project
